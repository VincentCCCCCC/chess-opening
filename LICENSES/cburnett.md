# cburnett chess pieces

This project includes the cburnett chess piece SVG set for the in-game board.

Source:

- https://github.com/lichess-org/lila/tree/master/public/piece/cburnett
- Download mirror used during local setup: https://cdn.jsdelivr.net/gh/lichess-org/lila@master/public/piece/cburnett/

Files:

- `miniprogram/assets/pieces/cburnett/wK.svg`
- `miniprogram/assets/pieces/cburnett/wQ.svg`
- `miniprogram/assets/pieces/cburnett/wR.svg`
- `miniprogram/assets/pieces/cburnett/wB.svg`
- `miniprogram/assets/pieces/cburnett/wN.svg`
- `miniprogram/assets/pieces/cburnett/wP.svg`
- `miniprogram/assets/pieces/cburnett/bK.svg`
- `miniprogram/assets/pieces/cburnett/bQ.svg`
- `miniprogram/assets/pieces/cburnett/bR.svg`
- `miniprogram/assets/pieces/cburnett/bB.svg`
- `miniprogram/assets/pieces/cburnett/bN.svg`
- `miniprogram/assets/pieces/cburnett/bP.svg`

Runtime PNGs:

- `miniprogram/assets/pieces/cburnett-png/*.png`
- These PNG files are local rasterized copies of the SVG files above, generated only because WeChat Mini Game Canvas image loading is more reliable with PNG than SVG.
